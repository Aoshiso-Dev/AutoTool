AutoTool ご利用ガイド（配布版）

このファイルは、AutoTool.exe と dll を受け取った利用者向けの説明です。

--------------------------------------------------
1. 使い始める前に
--------------------------------------------------
- AutoTool は Windows 用アプリです。
- この配布版は「フレームワーク依存配布」です。
  起動できない場合は .NET 8 Desktop Runtime のインストールが必要です。

--------------------------------------------------
2. 配布ファイルの置き方
--------------------------------------------------
- GitHub から取得した ZIP は、まず任意フォルダへすべて展開してください。
- AutoTool.exe と同じフォルダに、同梱されている dll 一式をそのまま置いてください。
- フォルダ構成を変えたり、dll を一部だけ移動/削除しないでください。

推奨例:
  C:\AutoTool\
    AutoTool.exe
    *.dll
    *.deps.json / *.runtimeconfig.json
    Readme.txt
    Readme_コマンド詳細.txt
    Settings\appsettings.json

--------------------------------------------------
3. 起動方法
--------------------------------------------------
- AutoTool.exe をダブルクリックして起動します。

起動しない場合:
- SmartScreen が表示されたら「詳細情報」→「実行」を選択
- .NET 8 Desktop Runtime をインストール後、再度起動

--------------------------------------------------
4. 基本操作（最短）
--------------------------------------------------
1. コマンドを追加して手順を作成
2. 必要な値を設定
3. .macro ファイルとして保存
4. 実行前チェック結果を確認（診断ボタンで表示切替）
5. 実行してログを確認

緊急停止:
- 実行中に Esc を約1.2秒長押しすると、マクロを緊急停止できます。
- この緊急停止は AutoTool が非アクティブな状態でも有効です。
- 実行中でも「停止」ボタンを押して停止できます。

実行前チェックについて:
- 実行ボタン押下時に、画像パス/OCR(tessdata)/外部実行ファイルパスを一括診断します。
- 要修正がある場合は実行を開始せず、チェック一覧にエラーコード付きで表示します。
- 一覧は画面右下の「診断」ボタンでいつでも開閉できます。
- 診断パネル右上の「×」ボタンで閉じられます。
- コマンドリストを操作（選択/追加/移動/削除/貼り付けなど）すると診断パネルは自動で閉じます。
- 実行エラー時はログに「復旧ガイド（原因候補/対処/次アクション）」が表示されます。
- お気に入りは「置換読込」に加えて「挿入」でテンプレートとして再利用できます。
- OCR系コマンドの編集時は「OCRプレビュー」で結果を事前確認できます。
- OCR系コマンドの「自動調整」で前処理/PSM/最小信頼度の推奨値を適用できます。
- AI検出系コマンドの編集時は「AI検出テスト」で結果を事前確認できます。
- AI検出系コマンドの「自動調整」で conf/iou の推奨値を適用できます。
- 画像検索系コマンドの「自動調整」で推奨しきい値を適用できます。
- OCR/画像検索テストで検知に成功すると、検知位置が赤枠で点滅表示されます。
- これらの自動調整結果は編集表示だけでなく、コマンド設定値にも即時反映されます（保存対象）。

--------------------------------------------------
5. データ保存場所
--------------------------------------------------
- マクロ: Macro\ または保存した *.macro ファイル
- 設定: Settings\appsettings.json
- 画面状態: Settings\window_settings.json

--------------------------------------------------
6. 更新版を上書き配布するときの注意
--------------------------------------------------
- 既存のユーザーデータを残すため、次は削除しないでください。
  - Macro\
  - Settings\
- 通常は AutoTool.exe と dll を新しいものに差し替えてください。

--------------------------------------------------
7. さらに詳しい資料
--------------------------------------------------
- Readme_コマンド詳細.txt（コマンド別の詳細説明）

--------------------------------------------------
8. よくあるトラブル
--------------------------------------------------
- 「dll が見つからない」と表示される:
  - 配布物一式を同じフォルダに展開し直してください。
- OCR が動かない/精度が低い:
  - 設定画面で tessdata ディレクトリを確認してください。
- 保存できない:
  - インストール先フォルダの書き込み権限を確認してください。
