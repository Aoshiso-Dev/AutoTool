# 利用者向けガイド

## 1. このアプリでできること

AutoTool は、繰り返し作業を「マクロ」として保存し、再実行するためのツールです。

- クリック、キー入力、待機などの操作自動化
  - クリック操作では「押下維持時間（ms）」を設定可能
  - クリック操作の注入方式は `MouseEvent` / `SendInput` から選択可能
  - クリック前のマウス移動シミュレートを ON/OFF で選択可能
- 画像検索（OpenCV）によるターゲット検出
- OCR（Tesseract）による文字判定
- AI 検出（ONNX Runtime）による対象検出

## 2. はじめに（初回セットアップ）

前提:

- Windows 環境
- .NET 8 SDK がインストール済み

実行:

```powershell
dotnet restore .\AutoTool.sln
dotnet build .\AutoTool.sln -c Debug
dotnet run --project .\AutoTool.Bootstrap\AutoTool.Bootstrap.csproj
```

## 3. 画面の見方

メイン画面は主に次の領域で構成されます。

- タイトルバー: 開く、保存、Undo/Redo
- マクロ編集エリア: コマンド一覧の編集
- ログパネル: 実行中のログ確認
  - `コピー`: 現在表示中（検索結果を含む）のログをクリップボードへコピー
  - `クリア`: ログ一覧を消去
- 実行前チェックパネル: 実行前の設定不備（画像/OCR/外部実行）確認
- お気に入りパネル: よく使う構成の呼び出し
  - `置換読込`: 一覧全体をテンプレートで置き換え
  - `挿入`: 選択行の下へテンプレートを追加挿入
- ステータスバー: 実行状態・メッセージ表示

ショートカット:

- `Ctrl+O`: 開く
- `Ctrl+S`: 上書き保存
- `Ctrl+Shift+S`: 名前を付けて保存
- `Ctrl+Z`: 元に戻す
- `Ctrl+Y` / `Ctrl+Shift+Z`: やり直し
- `Ctrl+C`: 選択中コマンドをコピー
- `Ctrl+V`: コピー済みコマンドを選択行の下へ貼り付け
- `Ctrl+X`: 選択中コマンドを切り取り（コピーして削除）
- `Alt+↑` / `Alt+↓`: 選択中コマンドを上下に移動
- `Ctrl+A` → `Delete`: コマンド一覧を全削除
- `Shift+クリック` / `Ctrl+クリック`: コマンド一覧の複数選択
  - 複数選択した状態で `Delete` / `Ctrl+X` / `Ctrl+C` / `Ctrl+V` が使えます

設定:

- タイトルバー右側の `設定` ボタンから設定画面を開けます
- 「前回終了時の状態を次回起動時に復元する」を ON/OFF できます
- 「メインウィンドウサイズ」を `S/M/L` から選択できます
  - `S`: 作業領域の 30%
  - `M`: 作業領域の 50%
  - `L`: 作業領域の 70%
  - 初回起動時は表示領域に応じて自動選択（4K 相当は `L`、それ以外は `M`）
- コマンドリストはウィンドウ幅に応じて表示密度を自動調整します
  - コンパクト時: 説明列を隠して操作列を優先表示
  - 標準/広め: 説明列を表示し、行高・列幅を拡張
  - `S/M/L` に応じて行高とフォントサイズを切り替えます（`S` は密度優先、`L` は視認性優先）
- コマンド操作ボタン列に `コピー` / `貼り付け` ボタンがあります（ショートカットと同等）
- 画面右下の `診断` ボタンで、実行前チェックパネルを開閉できます
- OCR系コマンド（テキスト検索/Ifテキスト/OCR変数設定）では編集パネルに `OCRプレビュー` が表示されます
- 画像検索系コマンド（待機/クリック/If/画像検索）では編集パネルに `画像検索テスト` が表示されます
- AI検出系コマンド（If/クリック/変数設定）では編集パネルに `AI検出テスト` が表示されます

## 4. 基本操作フロー

1. コマンドを追加して手順を作る
2. 必要なパラメータを編集する
3. 保存して `.macro` ファイルとして管理する
4. 実行前チェック結果（要修正/問題なし）を確認する
5. 実行してログで結果を確認する

実行前チェック:

- 実行ボタン押下時に、画像パス・`tessdata`・外部プログラムパスの存在確認を一括実施します
- 要修正がある場合は実行を開始せず、チェック一覧に `[エラーコード]` 付きで表示します
- チェック一覧は実行前チェックパネルから再確認できます

テンプレート挿入:

- お気に入りへ追加した内容はテンプレートとして再利用できます
- `挿入` を押すと、現在のリストを置換せずに選択位置へ追加されます

OCRプレビュー:

- OCR系コマンド選択時に `テスト実行` で、その設定値（範囲/言語/PSM/前処理/tessdata）を使って単発OCRを実行します
- `自動調整` で前処理/PSM/最小信頼度の推奨値を試行し、最良候補を適用できます
- 結果文字列、信頼度、処理時間が編集パネル上に表示されます
- テスト成功時は検知範囲が赤枠で点滅表示されます
- `自動調整` 適用結果は編集UIだけでなく、選択中コマンドの設定値にも即時反映されます（そのまま保存対象）

画像検索テスト:

- 画像検索系コマンド選択時に `テスト実行` で、現在設定（画像/しきい値/探索対象）で単発検索を実行します
- `自動調整` でしきい値候補を順に評価し、推奨しきい値を適用できます
- 一致時はヒット座標と処理時間を表示し、テンプレート範囲を赤枠で点滅表示します
- 不一致時は復旧ヒント（しきい値調整、探索対象確認、テンプレ再取得）が表示されます
- `自動調整` 適用結果は編集UIだけでなく、選択中コマンドの設定値にも即時反映されます（そのまま保存対象）

AI検出テスト:

- AI検出系コマンド選択時に `テスト実行` で、現在設定（モデル/しきい値/対象ウィンドウ）で単発検出を実行します
- `自動調整` で conf/iou の候補を評価し、推奨しきい値を適用できます
- 一致時は検出数・スコア・検出矩形を表示し、検出範囲を赤枠で点滅表示します
- 不一致時は復旧ヒント（クラスID確認、しきい値調整、対象ウィンドウ確認）が表示されます
- `自動調整` 適用結果は編集UIだけでなく、選択中コマンドの設定値にも即時反映されます（そのまま保存対象）

エラー復旧ガイド付き実行ログ:

- 実行エラー時はログに「復旧ガイド」「原因候補」「対処」「次アクション」を自動表示します
- `次アクション` に表示される行番号（例: `#05`）のコマンドを選択し、設定修正後に再実行してください

## 5. ファイルと設定

- マクロファイル拡張子: `.macro`
- 設定ファイル: `Settings\appsettings.json`
- ウィンドウ状態ファイル: `Settings\window_settings.json`
  - 終了時にウィンドウ位置/サイズ、サイズプリセット、タブ状態、パネル開閉状態、右パネル幅、最後に開いていた `.macro` を保存
  - 起動時に保存内容を自動復元（対象ファイルが存在しない場合は復元をスキップ）
- セッション復元設定: `Settings\appsettings.json`
- 最近使ったファイル一覧: `Settings\RecentFiles_*.xml`
- お気に入り一覧: `Settings\favorites.xml`
  - `UiState.RestorePreviousSession` を `false` にすると、起動時の「前回タブ状態/パネル状態/最後に開いていたファイル」の復元を無効化
  - 同じ設定は設定画面からも変更できます

## 6. よくあるトラブル

- 起動しない:
  - `dotnet --info` で .NET 8 SDK を確認
  - `dotnet build` のエラーを解消
- 画像/OCR が期待通り動かない:
  - 対象画面の解像度・表示倍率・類似度しきい値を調整
  - OCR コマンドの `tessdataディレクトリ` で `公式ページ` から取得元を確認できます
  - `jpn+eng取得` を押すと `jpn.traineddata` / `eng.traineddata` を自動取得できます
- 既定では実行ファイル配下の `.\Settings\tessdata` が使用されます
- 保存できない:
  - 書き込み先フォルダ権限を確認
